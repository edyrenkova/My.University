Випускна робота з теми: "The Helper For The US Student".
Виконала: Диренкова Емілія, учениця 11-В класу.
Керівники: Бондік І.Г., Зуєва М.Г.

Програма є електронним допоміжником студента американського ВНЗ, написаний мовою С# у середовищі Microsoft Visual Studio 2015, що є надійним персональним асистентом для користувача.

Програмно-апаратні вимоги:
	Процесор з тактовою частотою 1,6 ГГц або більше.
	350 Мб ОЗУ.
	58 MB вільного простору на диску.
	Відеоадаптер з підтримкою DirectX 9 (роздільна здатність 1366 х 768 або вище).
	Доступ до інтернету (необов'язково).

Підтримувальні операційні системи:
	Windows 10
	Windows 8.1
	Windows 8
	Windows 7 з пакетом оновлень 1 (SP1)
	Windows Server 2012 R2
	Windows Server 2012
	Windows Server 2008 R2 з пакетом оновлень 1 (SP1)
